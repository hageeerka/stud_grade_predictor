# Student Grade Predictor 

Проект для прогнозирования успеваемости студентов на основе их академических данных.

## Описание

Система анализирует исторические данные студентов (оценки, посещаемость и другие показатели) и прогнозирует их будущую успеваемость. Основные функции:

- Предобработка образовательных данных
- Построение моделей машинного обучения для прогнозирования
- Визуализация результатов
- Категоризация студентов по уровням успеваемости
